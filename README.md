sony-vpc-brightness-linux
=========================

Class to change brightness in Sony VPC Series notebook`s in Linux.

How to test:
-------------

    $ python -m doctest bright.py

Usage:
-------------

    $ python bright.py 50
    $ python bright.py up
    $ python bright.py down
    $ python bright.py max
    $ python bright.py min
